## MVP (Must have по MoSCoW)
- Онбординг

    1. Регистрация (GitHub/Google)
    2. Согласие на обработку данных
    3. Диагностика (5 вопросов -> стартовый уровень тревожности)

- Основной цикл

    4. Дневник мыслей (таблица КПТ: ситуация -> мысль -> эмоция -> переформулировка)
    5. Базовая экспозиция (1 упражнение)
    6. Анимация +XP и повышение уровня аватара

- Прогресс

    7. Дашборд (график снижения тревожности)
    8. Шкала уровня персонажа

- Удержание

    9. Push-уведомления (ежедневное напоминание)

- Монетизация

    10. Триал 3 дня -> предложение подписки
